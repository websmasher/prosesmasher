# Plan 04: Check trait + runner + first 3 checks

## Summary

Implemented the Check trait, runner, and first 3 checks: word-count, em-dashes, banned-words. Uses low-expectations for expectation suite pattern. Tests use hand-built Document structs (no parser dependency).

## Decisions

- **Check trait with `'static` str returns**: `id()`, `label()`, `supported_locales()` return `&'static str`/`&'static [Locale]` per clippy pedantic.
- **BoxedCheck type alias**: `pub type BoxedCheck = Box<dyn Check>` to avoid complex type warnings on `all_checks()`.
- **Locale filtering in runner**: `None` supported_locales means all locales. Check is skipped if locale not in list.
- **Em-dash check recurses into blockquotes**: counts em-dashes in paragraph text inside blockquotes too.
- **Banned words check recurses into blockquotes**: collects words from paragraphs inside blockquotes. Skips list items and code blocks.
- **Empty banned list skips check**: no expectation added if banned_words config is empty.
