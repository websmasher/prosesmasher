# --check flag for individual check filtering

## Summary

Added `--check` flag to filter by individual check IDs (comma-separated). Composes with `--group`: group narrows first, then `--check` narrows further. Unknown check IDs produce an error listing all available IDs.

## Usage

```bash
prosesmasher check file.md --check banned-words,em-dashes
prosesmasher check file.md --group terms --check banned-words
prosesmasher check file.md --check word-count --format json
```
